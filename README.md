# Just Scrub — Starter Website

A simple, fast static website for **Just Scrub** (car wash).

## Pages
- `index.html` — home, pricing, booking section
- `about.html` — about page

## Booking integration (choose one)
- **Setmore** (free widget embed) — recommended for a small service business
- **Google Calendar appointment schedule** (simple if you're already on Google)
- **Calendly** (popular; embed/link)

When you choose a tool, replace the placeholder in the Booking section.

## Deploy (GitHub Pages)
After pushing to GitHub:
1. Repo → Settings → Pages
2. Source: `main` branch / root

Then your site will be live on the GitHub Pages URL.
